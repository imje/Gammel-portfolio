Portfolio for Photography

A simple website to showcase photos

Project description

The goal of this project was to make a portfolio website for a photographer. We wanted the photos to be the focus on the website, so we kept the colors and design to a minimal. To give the website a unique look we added some "fancy" functionality to the navigation bar.

Team

This project was done in collaboration with my group in the course "webteknologi" at NTNU.
